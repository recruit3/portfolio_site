MAIN PROJECT

AI同士会話システム（ai_chat）

ローカルLLMを用いて、AI同士が自動で会話を継続するシステム。

概要

人間を介さず、AI同士が自律的に会話を生成する仕組みを実装。 LLMの挙動・会話生成の特性を検証するためのプロジェクト。

背景・課題

  • AIは通常「人間との対話」が前提
  • AI同士の会話挙動を検証したい
  • プロンプト設計の影響を理解する目的

工夫した点

  • AI→AIのプロンプト受け渡し設計
  • 無限ループ防止（回数制御）
  • キャラクター設定による会話変化
  • 会話ログの保存と可視化

使用技術

  • Python / Django
  • Ollama（ローカルLLM）
  • JavaScript

課題・改善点

  • 会話の一貫性向上
  • 意味のある対話生成
  • 処理負荷の最適化
