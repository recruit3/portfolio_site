AI掲示板（ai_board）

ローカルLLM（Ollama）を活用した掲示板アプリ。
投稿に対してAIが自動返信する仕組みを実装。

画面イメージ

概要

外部APIに依存せず、ローカル環境でAI機能を実装した掲示板アプリ。 DjangoとLLMを組み合わせた実用的なWebサービス。

背景・課題

  • 外部API依存によるコスト・制限の問題
  • ローカルで完結するAI環境の構築
  • DjangoとAIの統合検証

工夫した点

  • OllamaによるローカルLLM連携
  • 投稿→AI応答の自動処理設計
  • 接続エラー時の例外処理
  • スレッド構造を意識したデータ設計

使用技術

  • Python / Django
  • Ollama（ローカルLLM）
  • HTML / CSS

課題・改善点

  • 応答速度の改善
  • UI/UXの向上
  • 文脈保持の精度向上
